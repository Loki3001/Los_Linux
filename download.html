üöÄ Installation Roadmap

Step 1: Prepare Boot Media

Option A: Ventoy (Recommended)

  1. Download Ventoy
  2. Install Ventoy to USB drive
  3. Copy ISO file to USB (no flashing needed!)

Option B: Rufus (Windows)

  1. Download Rufus
  2. Select LokeshOS ISO
  3. Choose GPT partition scheme
  4. Click Start

Option C: dd (Linux/Mac)

sudo dd if=LokeshOS-Pro.iso of=/dev/sdX bs=4M status=progress
sudo sync

Step 2: Configure BIOS

For Lenovo ThinkBook 14-IML:

  1. Shut down laptop completely
  2. Press Novo Button (small button near power)
  3. Select BIOS Setup
  4. Go to Boot tab
  5. Set Boot Mode to UEFI
  6. Disable Secure Boot (if enabled)
  7. Save & Exit (F10)

üí° Tip: Write down current settings before changing

Step 3: Boot & Install

  1. Insert USB drive
  2. Press Novo Button ‚Üí Boot Menu
  3. Select USB device
  4. Choose "Install LokeshOS Pro" from GRUB menu
  5. Follow on-screen installer
  6. Create user account & password
  7. Wait for installation (~15 minutes)
  8. Reboot & remove USB
