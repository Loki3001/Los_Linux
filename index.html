üöÄ Installation Guide

Step 1: Create Bootable USB

Windows:

  1. Download Rufus or Ventoy
  2. Select LOS ISO file
  3. Choose USB drive (8GB+)
  4. Click Start/Install

Linux:

sudo dd if=LOS.iso of=/dev/sdX bs=4M
sudo sync

macOS:

  1. Download balenaEtcher
  2. Select LOS ISO
  3. Select USB drive
  4. Click Flash

Step 2: Boot from USB

For ANY PC/Laptop:

  1. Insert USB drive
  2. Restart computer
  3. Press Boot Menu key during startup:
    • Dell: F12
    • HP: F9 or ESC
    • Lenovo: F12 or Novo button
    • ASUS: F8 or ESC
    • Acer: F12
    • Others: Usually F12, F9, or ESC
  4. Select USB device from boot menu

Step 3: Install LOS

  1. Choose "Install LOS" from boot menu
  2. Select language & keyboard
  3. Choose installation type:
    • Erase disk (full install)
    • Manual partition (dual-boot)
  4. Create user account & password
  5. Wait 10-15 minutes for installation
  6. Reboot and remove USB
  7. Enjoy LOS!

‚ö†Ô∏è Important Notes

  • Backup your data! Installation will erase the target drive
  • LOS works on all PC brands - Dell, HP, Lenovo, ASUS, Acer, custom builds, etc.
  • Both UEFI and Legacy BIOS modes supported
  • First boot takes 2-3 minutes (automatic hardware detection)
  • Wi-Fi drivers load automatically for most adapters
